# ЁЯза UAS Kecerdasan Buatan

## Prediksi Risiko Stroke Menggunakan Algoritma Decision Tree dan K-Nearest Neighbor

### ЁЯУЦ Deskripsi Proyek

Proyek ini merupakan Tugas Besar Ujian Akhir Semester (UAS) mata kuliah **Kecerdasan Buatan**. Tujuan utama proyek adalah membangun model _Machine Learning_ untuk memprediksi kemungkinan seseorang mengalami **stroke** berdasarkan data demografi, kondisi kesehatan, dan gaya hidup pasien.

Model yang dibangun menggunakan dua algoritma klasifikasi, yaitu:

- Decision Tree
- K-Nearest Neighbor (KNN)

Selanjutnya kedua algoritma dibandingkan berdasarkan nilai Accuracy, Precision, Recall, dan F1-Score untuk mengetahui model dengan performa terbaik.

# ЁЯУВ Dataset

Dataset yang digunakan adalah **Stroke Prediction Dataset** yang berisi data kesehatan pasien untuk memprediksi kemungkinan terjadinya stroke.

Informasi dataset:

- Jumlah data : **5.110**
- Jumlah atribut : **12**
- Target : **stroke**
- Jenis masalah : **Binary Classification**

Struktur atribut utama:

- id
- gender
- age
- hypertension
- heart_disease
- ever_married
- work_type
- Residence_type
- avg_glucose_level
- bmi
- smoking_status
- stroke

Dataset diperoleh dari Kaggle dan digunakan hanya untuk keperluan akademik.
